A shortest path algorithm
219 1

An Introduction,

Dijkstra's algorithm is an efficient algorithm for solving the single-source shortest path problem in a connected, weighted graph. It works by using the concept of a "current node," which is the node from which the current shortest path is calculated. The algorithm starts with the current node being the source and then iteratively finds a shortest path to the other nodes in the current graph.
