How about the Floyd-Warshall algorithm? It is a recursive algorithm that can be used to find the shortest path between every pair of nodes in a graph. It works by performing three steps at each iteration: 1. Compute all paths between every node in the graph and its neighbors. 2. Compare each path's length with all previously calculated paths between the same nodes. 3. Store the shortest path if it is shorter than any previously stored paths. This algorithm runs in cubic time, which means that it takes O(n^3) to complete where n is the number of nodes in the graph.

A shortest path algorithm

An Introduction,

Dijkstra's algorithm is an efficient algorithm for solving the single-source shortest path problem in a connected, weighted graph. It works by using the concept of a "current node," which is the node from which the current shortest path is calculated. The algorithm starts with the current node being the source and then iteratively finds a shortest path to the other nodes in the current graph.
